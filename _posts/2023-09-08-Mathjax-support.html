Mathjax is a javascript engine for displaying \(\LaTeX\) commands on webpages. An inline equation should be put inside a pair of \ backslash and ( bracket. For example, \(f(x)\) produces \(f(x)\). Math equations that go on a single line should be place indside \ backslash and [ square bracket pair. For example, \[ f(x) - x < 0 \] produces the following. \[ f(x) - x < 0 \]

I am planning to add some sort of .sty file for this website. If the need arise, which certainly does, I will implement some extension for Mathjax.
